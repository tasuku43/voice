# Non-goals

MVP explicitly excludes:

- full IME replacement,
- App Store packaging,
- cloud sync,
- team dictionary administration,
- meeting recording,
- system audio capture,
- speaker diarization,
- real-time character-by-character dictation,
- automatic prompt submission,
- automatic command execution,
- cloud STT,
- uploading transcripts,
- LLM-based rewriting as the primary correction mechanism.

These may be revisited only after the local dictionary-learning loop proves useful.
